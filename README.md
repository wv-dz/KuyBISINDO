# Kuy BISINDO

Project media pembelajaran Bahasa Isyarat Indonesia (BISINDO) berbasis aplikasi Android hasil kolaborasi bersama Mahasiswa Pendidikan Khusus UNJ.


## Features
- Video pembelajaran terdiri dari :
    - Abjad
    - Buah
    - Hari & Bulan
    - Kata Ganti Orang
    - Percakapan
    - Cover Lagu
- Game Quiz
